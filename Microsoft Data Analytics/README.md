# Microsoft Learn – Power BI Data Analytics Learning Paths

## Overview

This repository documents my completion of multiple official Microsoft Learn learning paths focused on Data Analytics and Microsoft Power BI.

The training covers the full analytics lifecycle — from foundational data concepts and preparation to advanced data modeling, report design, governance, and security implementation.
