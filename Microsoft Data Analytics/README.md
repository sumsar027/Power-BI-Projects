# Microsoft Learn – Data Analytics Learning Path

## Overview
This repository documents my completion of the Microsoft Learn Data Analytics learning path. The program focuses on core data analytics concepts and Microsoft tools such as Power BI and Microsoft Fabric.

## Skills Gained
- Data analysis principles and workflows  
- Building interactive dashboards in Power BI  
- Data modelling and transformation  
- End-to-end analytics using Microsoft Fabric  
- AI-assisted analytics with Copilot in Power BI  

## Outcome
Through this learning path, I developed the ability to transform raw data into actionable insights using Microsoft’s analytics ecosystem.
