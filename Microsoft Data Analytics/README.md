Microsoft Learn – Power BI Data Analytics Learning Paths
Overview

This repository documents my completion of multiple official Microsoft Learn learning paths focused on Data Analytics and Microsoft Power BI.

The training covers the full analytics lifecycle — from foundational data concepts and preparation to advanced data modeling, report design, governance, and security implementation.

The image below shows the completed learning paths and their estimated duration.

Learning Path Progress

Learning Paths Completed
1. Get Started with Microsoft Data Analytics

Duration: 1 hr 44 min

Introduction to the role of a Data Analyst

Core data analytics concepts

Overview of Microsoft’s analytics ecosystem

End-to-end analytics workflow

This module establishes the theoretical foundation for working with data in business environments.

2. Prepare Data for Analysis with Power BI

Duration: 4 hr 33 min

Connecting to multiple data sources

Data cleaning and transformation using Power Query

Data shaping and profiling

Handling inconsistencies and missing values

Focus: Ensuring data quality and reliability before modeling.

3. Model Data with Power BI

Duration: 6 hr 59 min

Data modeling best practices

Creating and managing table relationships

Writing calculated columns and measures using DAX

Optimizing model performance

Focus: Building scalable and efficient analytical models.

4. Design Effective Reports in Power BI

Duration: 5 hr 7 min

Data visualization principles

Report layout and UX best practices

KPI development

Advanced visualization techniques

Focus: Transforming structured data into actionable insights.

5. Manage and Secure Power BI

Duration: 2 hr 48 min

Workspace management

Dataset governance

Row-Level Security (RLS)

Deployment and sharing strategies

Focus: Enterprise-grade security and controlled distribution of analytics solutions.
